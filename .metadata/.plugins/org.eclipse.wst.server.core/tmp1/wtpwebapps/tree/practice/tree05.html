알고보면 진득한 창작자들은 거진 다 똘기라는 걸 가지고 있다. 심지어 이번에 소개할 저작자는 딴지일보 기자였단다.(나무위키 출저) 무려 초창기 멤버란다. (굉장히 디테일하고 멋지게(?) 기술되어있다. 그래서 이거 혹시 저작자 자신이 작성한게 아닌가 하는..)

아무리 가장 미천하고 황당한 입소문에 의해서 만들어진 것이든 혹은 공신력 있는 어떤 매체에 의해서 입소문이 만들어진 것이든 남의 입에서 구전으로 올라오거나 내려오는 콘텐츠는 언제나 관심이 가기 마련이다.

보통 그러한 콘텐츠는 직접 체험을 하더라도 기대를 하거나 혹은 실망을 생각하며 본다. 그러기 때문일까? 의외로 그렇게 좋지도 그렇게 나쁘지도 않는 정규분포라는 결론을 내려버린다. ‘별것도 아닌거 가지고 서로 지랄들이네’라는 문장으로 함축될만하게 니들의 객관적인 지표, 객관적인 생각따윈 개나 줘버리고 하며 등을 돌려버린다. 관심밖으로 사라지고 관계가 끝나버린다. 알고보니 그게 그렇더라는 직접체험의 결론이 보통 그렇게 끝이난다. 난 그렇다.
