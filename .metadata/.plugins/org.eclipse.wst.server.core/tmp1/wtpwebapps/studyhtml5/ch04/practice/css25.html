box-sizing:borderbox먹이고 display: inline-block; div 안에 width 30%와 70% 이다.
※중요한점은 inline-block은 코드 정렬 따위로 스페이스바(띄어쓰기)가 어쩔 수 없이 먹혀질 수도 있다. 따라서 중간에 공간이 있다. 결국 이 공간으로 인해 회색 box width를 69%로 만들었다.그렇지 않으면 회색이 밑으로 밀려난다.
