notoSans폰트
및 웹 폰트를 사용할 때는 저작권 문제가 발생할 수 있기 때문에 무료로 사용을 원할 경우 반드시 무료로 공개된 것인지를 확인한다.무료로 폰트를 제공하는 사이트에는 구글폰트(http://www.google.com/get/noto/#/family/noto-sans-kore), 서울서체(http://www.seoul.go.kr/v2012/seoul/symbol/font.html)가 있다. 웹 폰트를 사용하려면 먼저 @font-face를 사용해서 사용할 폰트명(font-family)과 폰트파일명(src)을 정의한다. 그런 후 정의된 웹 폰트명을 적용할 태그에 font-family속성을 사용해서 쓴다.
