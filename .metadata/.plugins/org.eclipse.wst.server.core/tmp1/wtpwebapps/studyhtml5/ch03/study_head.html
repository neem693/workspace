head

head는 문서에 대한 정보를 써 놓은 곳으로 이 정보는 메타데이터(데이터를 설명하는 데이터)로 표현한다.

title style meta link script base등을 사용한다.

title

이건 설명안해도 알것이다.

base

웹 페이지상의 모든 링크를 위한 기본 주소 또는 기본 대상을 지정한다.

link

웹 페이지와 외부 자원간의 관계를 정의하며, 주로 스타일 시트 파일(CSS파일)링크에 사용한다.

head
link rel ="stlesheet" href = "main.css"
/head

meta

html 문서에 대한 메타데이터를 정의하며 주로 페이지 설명, 키워드, 인코딩 , 뷰포트, 거맥 로봇 차단등을 설정 할 때 사용한다.

페이지 설명

meta name = "description" 로 시작하며 보통 웹 페이지나 웹 사이트에 대한 개괄적인 설명을 기술한다.

head
meta name ="description" content = "테스트를 위한 사이트로..."
/head

키워드 사용

meta name = "keywords" 로 시작하며 검색 엔진 등에서 키워드 검색에 사용할 단어들을 content 속성 값에, (쉽표)를 써서 나열한다.

head
meta name ="keywords" content = "HTML5, CSS3, JavaScript"
/head

인코딩 설정

meta chareset ="인코딩 형식" 로 시작하여 charset 속성의 값으로 인코딩 형식을 지정해서 웹 페이지에 표시되는 문자의 인코딩을 설정한다. 인코딩 형식은 utf-8사용을 권장한다.

head
meta charset ="UTF-8"
/head

뷰포트 설정

meta name ="viewport"로 시작하며 content 속성의 값으로 화면의 내용 확대/축소 배율등을 지정한다. 부포트 설정은 웹 브라우저의 페이지 자동 크기 조정 기능을 중단 시킬 목적으로 사용한다. content 속성의 값을 "width=device-width,initial-cxale=1.0"과 같이 지정하면 원래의 크기대로 표시된다.

head
meta name = "viewport" content = "width=device-width,initial-scale=1.0"/
/head

뷰포트를 지정하는 (meta)태그 사용법

문법 meta name = "viewport" content="크기지정"/

크기 지정에 사용되는 속성 값
width : 뷰포트의 가로 너비, device-width를 값으로 가장 많이 사용
height : 뷰포트의 세로 높이, 일반적으로 잘 사용하지 않음
initial-scale: 내용의 확대/축소의 기본값을 지정. 0~10 사이의 값을 입력할 수 있으며, 전자정부 권고안은 원래크기인 1.0
maximum-scale: 내용을 확대할 수 있는 최대 값 지정. 0~10사이의 값을 입력할 수 있으며, 전자정부 권고안은 원래 크기인 1.0
minimum-scale: 내용을 확대할 수 있는 최소 값 지정. 0~10 사이의 값을 입력할 수 있으며, 전자정부 권고안은 원래크기인 1.0
user-scalable: 사용자가 손가락을 사용하여 콘텐츠를 직접 확대/ 축소할 수 있는 여부를 yes 또는 no로 지정, 기본값을 yes이며, 전자 정부 권고안은 no

사용예
(meta name = "viewport" content ="width=device-width,initial-scale=1.0"/)

가자

검색 로봇 차단

웹 페이지가 검색되지 않도록 검색 로봇의 접근을 차단할 때는 meta name = "robots" 로 시작한다. 구글 해킹을 방어하기 위해서는 구글 검색 로봇만을 차단할 때는 meta name ="googlebot"으로 시작한다.

모든 검색 로봇 차단

head
meta name = "viewport" content = "NOINDEX, NOFOLLOW"
/head

구글 검색 로봇 차단

head
meta name = "googlebot" content = "NOINDEX, NOFOLLOW"
/head

페이지 새로고침

<meta http-equiv= "refecsh">를 사용해서 페이지를 새로고침 하거나 특정 페이지로 이동시킬 수 있다.

30초마다 페이지 새로 고침

<head>
<meta http-equiv= "refresh" content = "30">
</head>

study.html 페이지로 이동

<head>
<meta http-equiv= "refresh" content = "0;url=study.html">
</head>

스크립트 태그

웹 페이지의 처리를 담당하는 코드인 자바스크립트 <script> 자바 스크립트(클라이언트 사이드 스크립트)를 정의한다. <script>와 </script>태그 사이에 자바스크립트 코드를 기술한다.

<script>태그 사용 - 화면에 "Test"라는 내용을 가진 메시지 상자를 표시

<head>
<script>
 function test(){
alert("test")
}
</script>
</head>

<style>태그

웹 페이지를 위한 스타일 저보를 정의하며 웹 브라우저에서 HTML 태그를 표현하는 방법을 기술한다.<style>과 </style>태그 사이에 스타일 시트 코드를 기술한다.

<style>태그 사용 - 웹 페이지 배경 색을 파랑으로 지정

<head>
<style>
 body{ background-color : blue;}
</style>
</head>
