테이블 태그는 표 영역만을 설정하는 태그이다.

표를 이루려면 행, 열을 표시하는 tr td 태그를 사용해야한다. 테이블 태그는 고정 크기의 웹 사이트를 개발하는 경우 사용하는 것이 권장된다.
반응형 웹 사이트와 같이 유동형 구조의 표현에는 거의 사용되지 않는다. 속성 태그의 표현과 관련된 것은 CSS파일에서의 지정하는 것을 권장한다.

주요속성

align = "left/right/center" : 전체 표가 정렬되는 위치 지정, 태그 표현
bgcolor = "색상코드" : 배경색, 태그 표현
background="배경이미지의 파일명" 배경이미지, 태그 표현
border ="n" : 칸과 칸 사이의 간격, 태그 표현
cellspacing="n" : 칸과 칸 사이의 간격, 태그 표현
cellpadding = "n" : 칸의 테두리선과 실제 안에 기록되는 내용과의 간격, 태그 표현
width = "n%" 혹은 "n" : 테이블의 너비, 태그 표현

tr 태그

행을 표시하는 태그. 속성은 CSS파일에서 지정하는 것을 권장함.

주요속성

align = "left | right |center" : 해당 행의 가로 정렬, 태그 표현
valign = "top | middle | bottom" : 해당 행의 세로 정렬, 태그 표현
height = "n%" 혹은 "n" : 해당 행의 높이. 행의 높이는 주로 <tr> 태그에서 지정함. 태그표현.

< td > 태그

표의 열 즉, 하나의 셀을 정의하는 태그이다. 정의하지 않은 속성은 상위 태그인 < table > 또는 < tr > 태그에서 상속 받는다. 속성 중 태그의 표현과 관련된 것은 CSS파일에서 지정하는 것을 권장한다.

주요속성

align = "left/right/center" : 해당 셀 안의 가로 정렬. 태그 표현
valign="top/middle/bottom": 해당 셀 안의 세로 정렬, 태그 표현 width="n%" 혹은 "n" : 해당셀의 너비. 태그 표현
colspan ="n": n으로 지정한 수만큼 열을 합침. 셀 합치기는 < td >태그에서 지정함.
rowspan = "n": n으로 지정한 수만큼 행을 합침. 셀 합치기는 < td >태그에서 지정함.
