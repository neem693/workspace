FTP 접속 사용법 안내


컴퓨터공학입문 실습에서 사용할 서버 주소와 FTP 사용을 다음과 같이 안내합니다.

1. FileZilla Client 프로그램을 다운로드 받아서 설치를 합니다.
FileZilla 이외에 다른 프로그램을 사용하여도 무방합니다.
다운로드 사이트 : https://filezilla-project.org/download.php

2. 설치한 FileZilla 프로그램을 실행하고 1번 화살표의 사이트관리자를 클릭합니다.

3. 사이트 관리자창에서 “새 사이트(N)” 버튼을 클릭하여 사이트를 추가합니다.

4. 사이트를 다음과 같이 등록합니다.

① 사이트 이름을 “컴공입문”

② 호스트 : html.flykorea.kr

③ 포트 : 10021

④ 로그인유형 : 일반

⑤ 사용자명 : cse학번

⑥ 비밀번호 : 메일 참조(분실시 교수님께 문의)

⑦ 입력을 다 하였으면 “연결” 버튼을 클릭하여 접속 여부 확인

5. 다음 그림과 같이 나오면 접속된 것입니다.

① 로컬 사이트는 내 컴퓨터입니다.

② 리모트 사이트는 원격지에 있는 html.flykorea.kr 서버입니다.

③ 서버의 public_html 폴더에 업로드하여야 내 홈페이지에 보여지게 됩니다..

6. 자신의 홈페이지 접속 방법은
http://html.flykorea.kr/~cse학번
예) http://html.flykorea.kr/~cse2016136001
