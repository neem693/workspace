기본 태그 내용
마크업 언어 HTML은 마크업 언어로 웹 문서를 작성하며, 태그를 사용해서 문서의 구조등을 기술하는 언어이다.
<!DOCTYPE> 현재의 문서를 HTML문서로 선언한다.
<html></html> <html>과 </html> 사이에 html 문서를 작성한다.
<head></head> <head>과 </head> 사이에 인코딩, 키워드, 뷰포트 등의 문서에 대한 정보를 제공한다.
<title></title> <title>과 </title> 사이에는 문서의 제목을 쓴다.
<body></body> <body>과 </body>사이에는 화면에 표시되는 문서의 내용을 쓴다.
<h1></h1> <h1>과 </h1>에는 제목 또는 주체를 쓴다.
<p></p> <p>와 </p> 사이에는 문단의 내용을 쓴다.
<img> 웹 페이지에 이미지를 삽일할 때 사용한다.
<a></a> 하이퍼링크(또는 링크)를 사용해서 이동할 페이지 또는 이동할 위치를 지정할 때 사용한다.
<tr><td> 테이블 관련 태그에서 <table> 태그는 표를 작성하는 태그이고 <tr>태그는 표의 행을 정의하고 <td>태그가 하나의 셀을 정의하는 태그이다.
<div></div> html문서에서 영역 또는 구역을 정의한다.
문서 구조 및 주요 태그 내용
HTML5 HTML5에서 문서의 타입, 인코딩 지정, 외부 CSS 링크 및 자바스크립트 코드 기술등에 사용되는 태그의 기술방법이 간단해지고 유연해졌다.
<header></header> 사이트의 소개나 네비게이션 등을 표시
<nav></nav> 사이트의 내비게이션(메뉴) 항목을 표시, 주로 <header> 태그에 넣어서 사용한다.
<article></article> 문서의 내용(글)을 표시하는 태그로, 주로 블로그의 글이나 기사와 같이 독립적인 콘텐츠를 표시
<aside></aside> 본문의 내용과는 독립적인 내용(주로 광고나 부 메뉴 등) 인사이드바 콘텐츠를 표시
<footer></footer> 사이트의 제작자, 주소, 저작권 정보등을 주로 표시
<details></details> 추가적인 정보나 사용자가 요청하는 정보를 표시
<summary></summary> <details> 엘리먼트의 머리글 표시
<dialog></dialog> 대화상자(dialog) 또는 창(window)을 정의한다. open속성을 사용해서 대화상자를 화면에 표시한다.
<figure></figure> 설명글을 붙일 대상을 정의
<figcaption></figcaption> <figure>태그로 정의한 대상에 설명글을 붙임. <figure>와 같이 사용됨
<mark></mark> 텍스트에 형광펜을 칠한 것과 같은 강조 효과를 갖는다.
<meter></meter> 측정값을 표시한다.
<progress></progress> 프로그래스 바를 표시한다.
<main></main> 사람이 읽을 수 있는 날짜나 시간을 표시하는 태그이다.
<time></time> 사람이 읽을 수 있는 날짜나 시간을 표시하는 태그이다.
<ruby></ruby> 루비 어노테이션(ruby annotation)을 기술한다.
<wbr> 어느 위치에 줄바꿈을 넣으면 좋은 지를 결정하는 태그이다.
<form></form> 사용자 입력을 묶어서 처리하는 것을 제공하며 실제로 사용자 입력을 웹폼 태그들을 사용한다. 웹 폼 태그에는 <input>, <select>, <textarea>, <button> 등과 html5에서 추가된 <datalist>,<keygen><output>등이 있다.
<input></input> 웹 폼 태그들 중 가장 많이 사용되는 것으로 텍스트 필드, 패스워드 필드, 체크 박스, 라디오 버튼, 파일 선택기, submit/reset/button등의 각종 버튼을 만들 때 사용하며, html5에서는 이메일 필드, URL 필드, 전화번호 필드, 숫자 필드, 날짜 선택 필드, 시간 선택 필드, 주 선택 필드, 색상 선택 등을 추가로 사용할 수 있다.
<select></select> 콤보 상자인 드롭다운 리스트를 만들 때 사용한다.
<textarea></textarea> 여러 줄을 입력할 수 있는 필드를 제공한다.
<button></button> 버튼을 만들 때 사용한다.
<datalist></datalist> 사용자가 텍스트 필드에 값을 입력할 때 선택할 수 있는 값의 목록을 제공한다.
<keygen></keygen> 양식을 서버로 전송할 때, 한 쌍의 키를 생성해서 private key(비밀키)는 로컬에 저장하고 public key(공용키)는 서버에 저장한다.
<output></output> 수학적 계산 결과 값을 표시한다.
<audio></audio> 오디오 파일을 재생할 때 사용된다.
<video></video> 비디오 파일을 재생할 때 사용된다.
<embed> 플래시 파일(.swf) 및 동영상 파일(.mp4 등), html파일 이미지 파일등을 플러그인 콘텐츠로써 화면에 표시한다.
<object></object> 웹 페이지에서 일종의 내장 객체로, 플래시 파일(.swf) 및 동여상 파일 html 파일, 이미지 파일, 등의 콘텐츠를 화면에 표시한다. 웹 페이지에서 자바애플릿, pdf리더, 플래시 플레이어와 같은 내장 플러그인을 쓸 대도 사용된다.
<iframe></iframe> 유튜브(youtube)동영상을 웹 페이지에 연결할 때 사용된다.
<canvas></canvas> 모든 종류의 그래픽을 그릴 수 있는 영역을 제공하며 주로 게임을 만들 때 사용된다.
<svg></svg>svg그래픽을 사용할 수 있는 영역을 제공한다.
