주요속성

src = "파일 이름" : 표시할 이미지 파일의 경로를 포함한 이름을 입력한다. 필수 속성으로 반드시 사용한다.
align = "left/right/top/middle/bottom": "left/right"은 글자가 같이 표시될 경우 그림이 표시되는 위치를 지정하고, "top/middle/bottom"은 그림과 같이 표시될 문자의 수직위치. 태그표현
border = "n" : 이미지 테두리의 두께. 태그표현
width="n": 이미지의 가로길이. 태그표현
height = "n" : 이미지의 세로길이. 태그표현
alt= "대신할 문자" 이미지가 표시되지 못할 경우 대신 표시할 메시지 지정
