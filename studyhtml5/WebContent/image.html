이미지

이미지는 웹문서의 구성의 중요한 부분이지만 text에 비해 요량이 크다. 다운로드 속도가 느려짐


사용가능한 3가지 포맷이 있다.
jpg:24bit color
gif:8bit color
png: 32bit color

jpg

web color의 특징: 발광체(스스로 빛을 내는)로서 rgb(red,green,blue)의 빛의 세기로 색상을 구현

각 원색은 2의 8승 (256단계, 8bit)단계로 밝기를 조정

rrggbb를 16진수로 표현

빨간색을 최대로 하는걸 16진수로 표현하면 ff0000

초록색을 최대로 하는걸 16진수로 표현하면 00ff00

파란색을 최대로 하는걸 16진수로 표현하면 0000ff

true컬러는 8비트 x 3 = 24bit 컬러 ->jpg

jpg는 압축률이 높음, 사진과 같이 세밀한 이미지에 적합하다.

gif

8비트 색 (256개의 색상만 가능), 로고, 심볼같은 단순한 이미지에 적합 컬러수가 적은것,특정 색상 하나를 투명한 효과를 줄 수 있음.

png

32bit 컬러로 투르컬러 + 알파 채널8bit로 투명도를 조정한다. 반투명 효과 (연기, 그림자) 적용에 적합하다.
